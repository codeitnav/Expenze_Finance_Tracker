### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/expenze.git
   cd expenze
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**  
   Go to `http://localhost:3000` to start using Expenze.
